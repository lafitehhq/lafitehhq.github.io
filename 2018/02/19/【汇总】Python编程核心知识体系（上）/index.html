对象

  1. 对象的概念
    Python 中的一切都可以赋值给变量或者作为参数传递给函数。【某些对象可以既没有属性也没有方法且不是所有的对象都可以子类化】

  2. 可变/不可变对象
    ①：可变对象【创建之后还可改变但是地址不会发生改变】： list【列表：使用[,]表示】, dict【字典：使用{‘’:’’,’’:’’}表示】, set【集合：使用set()表示】
    ②：不可变对象【创建之后不能更改，如果更改则变量会指向一个新的对象】：str, Number(int、float、bool、complex)【数字】, tuple【元组：使用(,)表示】 ）

  3. 对象的3个特征
    ①：身份：每个对象都有一个唯一的身份标识自己，任何对象的身份都可以使用内建函数 id() 来得到，可以简单的认为这个值是该对象的内存地址。

    ②：类型：对象的类型决定了对象可以保存什么类型的值，有哪些属性和方法，可以进行哪些操作，遵循怎样的规则。可以使用内建函数 type() 来查看对象的类型。

    ③：值：对象所表示的数据

  4. 对象的创建和引用

    对象是堆上分配的一个内存空间，用来表示对象所代表的值；变量是一个系统创建的表中的元素，拥有指向对象的引用；引用是从变量到对象的指针

上边的代码执行了以下操作：

②：如果变量 a 不存在，创建一个新的变量 a

  1. 共享引用
    ①：Python 中变量都是指向某一对象的引用，当多个变量都引用了相同的对象，成为共享引用。

②：但对于像列表这种可变对象来说则不同

③：对于字典和集合等没有分片概念的类型来说，可以使用 copy 模块中的 copy() 方法进行拷贝
  1. 对象相等

    == 操作符用于测试两个被引用的对象的值是否相等
    is 用于比较两个被引用的对象是否是同一个对象

  1. 对象的回收机制
    对象包含一个引用的计数器，计数器记录了当前指向该对象引用的数目，一旦对象的计数器为 0 ，即不存在对该对象的引用，则这个对象的内存空间会被回收。这就是 Python 中对象的回收机制，最明显的好处即在编写代码过程中不需要考虑释放内存空间。
    可以通过 sys 模块中的 getrefcount() 函数查询一个对象计数器的值

  2. 函数参数
    参数为可变数据对象：可能改变原对象
    参数为不可变数据对象：不改变原对象

内存管理

Python参数传递采用的是“传对象引用”的方式【相当于传值和传引用的一种综合】。
①：如果函数收到的是一个不可变对象（比如数字、字符或者元组）的引用，就不能直接修改原始对象——相当于通过”传值”来传递对象。
②：若函数收到的是一个可变对象（比如字典，列表或集合）的引用，就能修改对象的原始值——相当于通过“传引用”来传递对象。

  1. 编码问题

    2.1 位(bit)与字节(byte) 

    ①：1bit:2种可能    
②：1byte：8bit；256种可能

    2.2 ASCII；Unicode；utf-8(可变长)编码的关系 

    ①：ASCII编码是1个字节    
②：Unicode编码通常是2个字节  
③：UTF-8编码把一个Unicode字符根据不同的数字大小编码成1-6个字节，常用的英文字母被编码成1个字节，汉字通常是3个字节，生僻的字符才会被编码成4-6个字节。
④：Python2的默认编码方式是ASCII【使用u''.encoding('utf-8')转换】；Python3的默认编码方式是UTF-8

    2.3 记事本编辑时候： 

    从文件读取的UTF-8字符被转换为Unicode字符到内存里，编辑完成后，保存的时候再把Unicode转换为UTF-8保存到文件


    2.4.浏览网页的时候： 

    服务器会把动态生成的Unicode内容转换为UTF-8再传输到浏览器
